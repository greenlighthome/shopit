ShopIt is a light-weight, but fully functional web shop/application.
It implements all the latest technologies like Django framework, CSS3 (responsive design) and HTML5.
Capable of working on all types of devices.

ShopIt Web Application is shipped with its own python environment.

To install application:
$ pip install -U -r requirements.txt
